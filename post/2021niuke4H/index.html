题意

定义pip_{i}pi​表示第iii个素数，若x=∏piai y=∏pibix=\prod p_{i}^{a_{i}} \space y=\prod p_{i}^{b_{i}}x=∏piai​​ y=∏pibi​​，则定义运算x⊗y=∏pi∣ai−bi∣x \otimes y=\prod p_{i}^{|a_{i}-b_{i}|}x⊗y=∏pi∣ai​−bi​∣​，给定序列a1...na_{1...n}a1...n​，要求计算b1...nb_{1...n}b1...n​，其中bi=∑1≤j,k≤n,j⊗k=iajkcb_{i}=\sum_{1\leq j,k \leq n,j\otimes k=i}a_{j}k^{c}bi​=∑1≤j,k≤n,j⊗k=i​aj​kc​

输入：n,c,a1...n(1≤n≤106,0≤ai<998244353,0≤c≤109)n,c,a_{1...n}(1\leq n \leq 10^{6},0\leq a_{i}<998244353 ,0\leq c \leq 10^{9})n,c,a1...n​(1≤n≤106,0≤ai​<998244353,0≤c≤109)​​

思路

由题式子所得∏pimin(ai,bi)=gcd(x,y),∏pimax(ai,bi)=lcm(x,y)\prod p_{i}^{min(a_{i},b_{i})}=gcd(x,y),\prod p_{i}^{max(a_{i},b_{i})}=lcm(x,y)∏pimin(ai​,bi​)​=gcd(x,y),∏pimax(ai​,bi​)​=lcm(x,y)

则有x⊗y=∏pi∣ai−bi∣=lcm(x,y)gcd(x,y)=xygcd(x,y)2x \otimes y=\prod p_{i}^{|a_{i}-b_{i}|}=\frac{lcm(x,y)}{gcd(x,y)}=\frac{xy}{gcd(x,y)^{2}}x⊗y=∏pi∣ai​−bi​∣​=gcd(x,y)lcm(x,y)​=gcd(x,y)2xy​

设g=gcd(j,k);x=jg,j=xg;y=kg,k=ygg=gcd(j,k);x=\frac{j}{g},j=xg;y=\frac{k}{g},k=ygg=gcd(j,k);x=gj​,j=xg;y=gk​,k=yg

则bi=∑xy=i∑g=1min(nx,ny)axg(yg)cb_{i}=\sum_{xy=i}\sum_{g=1}^{min(\frac{n}{x},\frac{n}{y})}a_{xg}(yg)^{c}bi​=∑xy=i​∑g=1min(xn​,yn​)​axg​(yg)c

设m=min(nx,ny)m=min(\frac{n}{x},\frac{n}{y})m=min(xn​,yn​)

则∑g=1maxg(yg)c=ycxc∑g=1maxg(xg)c\sum_{g=1}^{m}a_{xg}(yg)^{c}=\frac{y^{c}}{x^{c}}\sum_{g=1}^{m}a_{xg}(xg)^{c}∑g=1m​axg​(yg)c=xcyc​∑g=1m​axg​(xg)c

复杂度：O(nlogn)

本题计算bib_{i}bi​可以先枚举xy=ixy=ixy=i，若是枚举xxx，则y∈[1,nx]y\in [1,\frac{n}{x}]y∈[1,xn​]，复杂度为1+n2+...+nx1+\frac{n}{2} +...+\frac{n}{x}1+2n​+...+xn​，即调和级数复杂度O(log(n))O(log(n))O(log(n))，反之枚举yyy的复杂度也是一样的

而计算bib_{i}bi​中后一项求和项可以枚举g∈[1,min(nx,ny)]g\in [1,min(\frac{n}{x},\frac{n}{y})]g∈[1,min(xn​,yn​)]，复杂度同样是O(log(n))O(log(n))O(log(n))

代码

#include <bits/stdc++.h>
#define INF 99999999
#define LINF LLONG_MAX 
using namespace std;
typedef long long ll;
typedef unsigned long long ull;
const int MAX_N=1000010;
const int mod=998244353;
//输入的n、m(c)，预处理i的m(c)次方，输入a数组，计算b数组(sum数组)，计算c数组(b数组)
int n,m,ans,pw[MAX_N],a[MAX_N],b[MAX_N],c[MAX_N];
//快速幂
int quickpow(int x,int n){
	int res=1;
	while(n){
		if(n&1)
			res=1ll*res*x%mod;
		x=1ll*x*x%mod;
		n>>=1;
	}
	return res;
}
//gcd
int gcd(int a,int b){
	if(b==0)
		return a;
	return gcd(b,a%b);
}

int main(){
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out","w",stdout);
	//输入
	cin>>n>>m;
	for(int i=1;i<=n;i++){
		cin>>a[i];
        //预处理1~n的m次方
		pw[i]=quickpow(i,m);
	}
	//枚举x
	for(int i=1;i<=n;i++){
		//枚举y
		//预处理b数组(sum数组)
		for(int j=1;i*j<=n;j++)
			b[j]=(b[j-1]+1ll*pw[j]*a[i*j]%mod)%mod;
		//计算c数组(b数组)
		for(int j=1;i*j<=n;j++){
			if(gcd(i,j)==1)
				c[i*j]=(c[i*j]+1ll*b[min(n/i,n/j)]*pw[j]%mod)%mod;
		}
	}
	//计算输出答案
	for(int i=1;i<=n;i++)
		ans^=c[i];
	cout<<ans<<'\n';
	return 0;
}

1 2 3 4 5 6
