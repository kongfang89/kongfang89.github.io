算法 | 快速傅里叶变换(Fast Fourier Transform, FFT)

2021 17th August 20:15 Tuesday 算法

快速傅里叶变换(FFT)可以加速多项式的乘法，也可以解决高精度乘法的问题(十进制数也可以视作多项式)

一、概念

**快速傅里叶变换(Fast Fourier Transform, FFT)**是一种加速多项式乘法的高效算法，

对于两个多项式y1=a0x0+a1x1+⋯+an−1xn−1,y2=b0x0+b1x1+⋯+bn−1xn−1y_{1}=a_{0}x^{0}+a_{1}x^{1}+\dots+a_{n-1}x^{n-1},y_{2}=b_{0}x^{0}+b_{1}x^{1}+\dots+b_{n-1}x^{n-1}y1​=a0​x0+a1​x1+⋯+an−1​xn−1,y2​=b0​x0+b1​x1+⋯+bn−1​xn−1​

若要计算y1×y2y1\times y2y1×y2​的结果：BF的复杂度是O(n2)O(n^{2})O(n2)，而FFT的复杂度是O(nlogn)O(nlogn)O(nlogn)​

前置知识：

  1. 多项式的表示法

    形如y=a0x0+a1x1+⋯+an−1xn−1y=a_{0}x^{0}+a_{1}x^{1}+\dots+a_{n-1}x^{n-1}y=a0​x0+a1​x1+⋯+an−1​xn−1​这样的多项式，我们有两种表示法可以唯一确定一个多项式

    • 系数表示法

      用多项式各项系数数组{a0,a1,…,an−1}\{a_{0},a_{1},\dots,a_{n-1}\}{a0​,a1​,…,an−1​}​​表示该多项式

    • 点值表示法

      在平面直角坐标系中，多项式函数yyy​可以被nnn​个点唯一确定，代入nnn​个点的坐标(x,y)(x,y)(x,y)​求解nnn​元方程组即可，即用{(x0,f(x0)),(x1,f(x1)),…,(xn−1,f(xn−1))}\{(x_{0},f(x_{0})),(x_{1},f(x_{1})),\dots,(x_{n-1},f(x_{n-1}))\}{(x0​,f(x0​)),(x1​,f(x1​)),…,(xn−1​,f(xn−1​))}​​表示该多项式

  2. 多项式相乘

      对于两个多项式各项系数数组{a0,a1,…,an−1},{b0,b1,…,bn−1}\{a_{0},a_{1},\dots,a_{n-1}\},\{b_{0},b_{1},\dots,b_{n-1}\}{a0​,a1​,…,an−1​},{b0​,b1​,…,bn−1​}​​​，多项式相乘需要系数逐个相乘，总共要计算n2n^{2}n2​​次

    • 点值表示法

      对于两个多项式：

      {(x0,f(x0)),(x1,f(x1)),…,(xn−1,f(xn−1))}\{(x_{0},f(x_{0})),(x_{1},f(x_{1})),\dots,(x_{n-1},f(x_{n-1}))\}{(x0​,f(x0​)),(x1​,f(x1​)),…,(xn−1​,f(xn−1​))}

    显然用点值表示法计算多项式相乘更为高效，而我们通常用系数表示法表示多项式，所以我们需要先将系数转点值，这里就用到了FFT算法，计算完后再用IFFT​​​​算法将点值转系数即可。

			for(int k=0;k<i;k++){
				//这里的x,y就是A0(x)和w*A1(x) 
				cp x=a[j+k];
				cp y=w*a[j+k+i];
				a[j+k]=x+y;
				a[j+k+i]=x-y;
				//w的上标从0开始，要跟着k走，所以每次乘一个wn使上标+1就行 
				w*=wn;
			}
		}
	}
	//如果是IFFT，则每一项再除以limit就行
	if(f==-1){
		for(int i=0;i<limit;i++)
			a[i]/=limit;
	}
}

int main(){
	freopen("1.in","r",stdin);
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out","w",stdout);
	//输入 
	cin>>s1>>s2;
	//获取字符串s1,s2的长度-1，也即最后一个数的下标 
	lena=strlen(s1)-1;
	lenb=strlen(s2)-1;
	//将两个数的倒序按位存在复数的实部中：比如123456789这个数，存在字符串中就是98765432100000......（下标由0到MAX_N-1） 
	for(int i=0;i<=lena;i++)
		a[i]=(double)(s1[lena-i]-'0');
	for(int i=0;i<=lenb;i++)
		b[i]=(double)(s2[lenb-i]-'0');
	//计算limit,len
	//limit:大于等于lena+lenb的第一个2的m次幂，这是为了便于FFT递归分治的实现 
	//len:log_{2}(limit)
	while(limit<lena+lenb){
		limit<<=1;
		len++;
	}
	//预处理出所有w_{n}{i}最终的位置 
	for(int i=0;i<limit;i++)
		rev[i]=(rev[i>>1]>>1)|((i&1)<<(len-1));
	//FFT:将a,b两个多项式的系数表示转为点值表示 
	FFT(a,1);
	FFT(b,1);
	//点值表示的多项式乘法：y值相乘 
	for(int i=0;i<=limit;i++)
		a[i]=a[i]*b[i];
	//IFFT:将多项式相乘结果的点值表示转为系数表示 
	FFT(a,-1);
	//将每一位进位后存到ans数组 
	for(int i=0;i<=limit;i++){
		ans[i]+=(int)(a[i].real()+0.5);
		ans[i+1]+=ans[i]/10;
		ans[i]%=10;
	}
	//数前面位为0的部分不要 
	while(limit&&!ans[limit])
		limit--;
	//输出答案 
	for(int i=limit;i>=0;i--)
		cout<<ans[i];
	cout<<'\n';
	return 0;
}
