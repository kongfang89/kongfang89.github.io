Codeforces Round #783 (Div. 2) D. Optimal Partition 题解

D. Optimal Partition

题意

给定nnn个数，要求将其分为若干个连续的非空子数组，使所有子数组的价值和最大。

数组aaa的价值定义为：

\begin{equation} val(a)=\left\{ \begin{aligned} r-l+1,s(l,r)>0 \\0,s(l,r)=0 \\ -(r-l+1),s(l,r)<0 \end{aligned} \right. \end{equation}

其中s(l,r)=al+⋯+ars(l,r)=a_l+\dots+a_rs(l,r)=al​+⋯+ar​

输入：t,n,a1…n(1≤t≤5×105,1≤n≤5×105,−109≤ai≤109)t,n,a_{1\dots n}(1\leq t \leq 5\times 10^5,1\leq n\leq 5\times 10^5,-10^9\leq a_i\leq 10^9)t,n,a1…n​(1≤t≤5×105,1≤n≤5×105,−109≤ai​≤109)

输出：分为若干个子数组后的最大可能的价值

思路

动态规划：

dp[i]dp[i]dp[i]表示前iii个元素的最大可能价值

  1. 暴力做法

    显然可以枚举每个iii，再枚举j(j<i)j(j<i)j(j<i)，统计以jjj为分界的价值更新答案。

    由于题意所给nnn较大，此法会超时。

    复杂度：O(n2)O(n^2)O(n2)

    dp[0]=0;
for(int i=1;i<=n;i++)
    for(int j=0;j<i;j++){
        if(s[i]-s[j]>0)
            dp[i]=max(dp[i],dp[j]+i-j);
        else if(s[i]-s[j]<0)
            dp[i]=max(dp[i],dp[j]+j-i);
        else
            dp[i]=max(dp[i],dp[j]);
    }
cout<<dp[n]<<'\n';

  2. 树状数组优化

    对于上述转移方程，我们可以变形为下列三个式子：

    dp[i]=max(dp[j]−j)+i,s[i]>s[j]dp[i]=max(dp[j]-j)+i,s[i]>s[j]dp[i]=max(dp[j]−j)+i,s[i]>s[j]

    dp[i]=max(dp[j]+j)−i,s[i]<s[j]dp[i]=max(dp[j]+j)-i,s[i]<s[j]dp[i]=max(dp[j]+j)−i,s[i]<s[j]

    dp[i]=max(dp[j]),s[i]=s[j]dp[i]=max(dp[j]),s[i]=s[j]dp[i]=max(dp[j]),s[i]=s[j]

    则我们可以用三个树状数组分别维护dp[j]−j,dp[j]+j,dp[j]dp[j]-j,dp[j]+j,dp[j]dp[j]−j,dp[j]+j,dp[j]的最大值，每次枚举iii时取三者最大值作为答案即可。

    由于三个式子均有条件限制，我们可以将前缀和数组sss排序，每次找到当前前缀和的位置，左区间即式子1，右区间即式子2，当前位置即式子3。

    注意：由于sss中的数过大，排序后需要离散化优化（因为s[i]=s[j]s[i]=s[j]s[i]=s[j]时只需要更新一次）。

    复杂度：O(nlog(n))O(nlog(n))O(nlog(n))

代码

#include <bits/stdc++.h>
#define INF INT_MAX
#define LINF LLONG_MAX 
#define endl '\n' 
using namespace std;
typedef long long ll;
typedef unsigned long long ull;
const int MAX_N = 5e5+10;

int T;
//m为离散化后的前缀和数组大小 
ll n,m;
//输入数组，离散化后的前缀和数组，前缀和数组 
ll a[MAX_N],s[MAX_N],ss[MAX_N];
//三个树状数组：第一个维护dp[i]+i的最值，第二个维护dp[i]-i的最值，第三个维护dp[i]的最值 
ll m1[MAX_N],m2[MAX_N],m3[MAX_N];
//dp[i]表示前i个元素的最大答案 
ll dp[MAX_N]; 

ll lowbit(ll x){
	return x&-x;
} 
//更新树状数组最值
//id表示当前前缀和的排位，也即1~id-1均小于id，id+1~m均大于id
//x即dp[i]，l即i
void modify(ll id,ll x,ll l){
	//左区间是前缀和比当前小的，即式1 
	for(int i=id-1;i;i-=lowbit(i))
		m1[i]=max(m1[i],x+l);
	//右区间是前缀和比当前大的，即式2 
	for(int i=id+1;i<=m;i+=lowbit(i))
		m2[i]=max(m2[i],x-l);
	//更新dp[i]最值 
	m3[id]=max(m3[id],x);
}
//查询max(dp[j]-j+i,dp[j],dp[j]+j-i)
ll query(ll id,ll r){
	ll res=m3[id];
	for(int i=id;i<=m;i+=lowbit(i))
		res=max(res,m1[i]-r);
	for(int i=id;i;i-=lowbit(i))
		res=max(res,m2[i]+r);
	return res;
}

int main(){
	freopen("1.in", "r", stdin);
	//ios_base::sync_with_stdio(false);
	//cout.tie(0);
	//cin.tie(0);
	//freopen("1.out", "w", stdout);
	cin>>T;
	while(T--){
		cin>>n;
		a[0]=0;
		s[0]=0;
		for(int i=1;i<=n;i++){
			cin>>a[i];
			//前缀和 
			ss[i]=a[i];
			ss[i]+=ss[i-1];
		}
		//离散化前缀和ss为s
		for(int i=0;i<=n;i++)
			s[i+1]=ss[i];
		sort(s+1,s+n+2);
		m=1;
		for(int i=2;i<=n+1;i++)
			if(s[i]!=s[i-1])
				s[++m]=s[i];
		//树状数组初始化 
		for(int i=1;i<=m;i++)
			m1[i]=m2[i]=m3[i]=-1e9;
		//dp 
		for(int i=0;i<=n;i++){
			//找到第一个大于等于ss[i]的值在s的位置，作为下标 
			int pos=lower_bound(s+1,s+m+1,ss[i])-s;
			//dp[0]=0跳过 
			if(i)
				//计算 
				dp[i]=query(pos,i);
			//更新 
			modify(pos,dp[i],i);
		}
		cout<<dp[n]<<'\n';
	}
	return 0;
}
