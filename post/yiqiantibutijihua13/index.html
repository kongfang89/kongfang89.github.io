D. For Gamers. By Gamers.

题意

你有nnn种战士和CCC元，每种战士有属性薪水、生命值、攻击力ci,hi,dic_i,h_i,d_ici​,hi​,di​，给定mmm次查询，每次给出一个BOSS生命值和攻击力H,DH,DH,D，问击败它需要至少需要多少代价，如果超过CCC，则输出−1-1−1

击败条件：hD>Hd,h=∑hi,d=∑di\frac{h}{D}>\frac{H}{d},h=\sum{h_i},d=\sum{d_i}Dh​>dH​,h=∑hi​,d=∑di​

输入：n,C,c,d,h,m,H,D(1≤n≤3×105,1≤C≤106,1≤c≤C,1≤d,h≤106,1≤m≤3×105,1≤D≤106,1≤H≤1012)n,C,c,d,h,m,H,D(1\leq n\leq 3\times 10^5,1\leq C \leq 10^6,1\leq c\leq C,1\leq d,h\leq10^6,1\leq m\leq 3\times 10^5,1\leq D\leq10^6,1\leq H\leq 10^{12})n,C,c,d,h,m,H,D(1≤n≤3×105,1≤C≤106,1≤c≤C,1≤d,h≤106,1≤m≤3×105,1≤D≤106,1≤H≤1012)

输出：击败每个怪物的代价，如果超过预算输出−1-1−1

思路

二分+贪心

根据hD>Hd\frac{h}{D}>\frac{H}{d}Dh​>dH​移项可得hd>HDhd>HDhd>HD，则对于每个士兵，我们可以将其生命值和攻击力相乘作为一项属性值，再预处理出花费[1,C][1,C][1,C]的最大伤害，对于每个查询二分查找答案即可。

复杂度：O(n+(m+C)logC)O(n+(m+C)logC)O(n+(m+C)logC)

代码

#include <bits/stdc++.h>
#define INF INT_MAX
#define LINF LLONG_MAX 
#define endl '\n' 
#define YES cout<<"YES"<<endl
#define NO cout<<"NO"<<endl
using namespace std;
typedef long long ll;
typedef unsigned long long ull;
const int mod= 1e9+7;
const int MAX_N = 1e6+10;

int n,c,m;
ll maxv[MAX_N];

int main(){
	freopen("1.in", "r", stdin);
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out", "w", stdout);
	cin>>n>>c;
	for(int i=1;i<=n;i++){
		ll co,x,y;
		cin>>co>>x>>y;
        //求出每个代价可以造成的最大伤害
		maxv[co]=max(maxv[co],x*y);
	}
    //更新一下每个代价的倍数情况（请多个战士）
	for(int i=1;i<=c;i++)
		for(int j=2*i;j<=c;j+=i)
			maxv[j]=max(maxv[j],maxv[i]*(j/i));
    //前一个代价的伤害<=后一个代价的伤害
	for(int i=1;i<=c;i++)
		maxv[i]=max(maxv[i],maxv[i-1]);
	cin>>m;
	while(m--){
		ll x,y;
		cin>>x>>y;
		ll h=x*y;
		if(h>=maxv[c])
			cout<<-1<<' ';
		else
			cout<<upper_bound(maxv+1,maxv+c+1,h)-maxv<<' ';
	}
	cout<<'\n';
	return 0;
}
