Codeforces Round #796 (Div. 2) E. Railway System 题解

E. Railway System

题意

给定一图有nnn个节点mmm条边，每次交互可以输出一个长度为mmm的010101串sss，若si=1s_i=1si​=1则第iii条边激活，若si=0s_i=0si​=0则第iii条边失效。对于每次交互，会返回一个值，表示图中mmm条边在sss对应的状态下的最大生成森林边权和。求该图对应的最小生成森林。最多查询2m2m2m次。

输入：? s?\space s? s——表示图中mmm条边的状态与sss对应，返回此时图的最大生成森林边权和

输出：! ans!\space ans! ans——表示图的最小生成森林边权和

思路

先查出每条边的权值，再利用克鲁斯卡尔求最小生成树的边权和。

首先记录图的最大生成森林边权和为lastlastlast。循环每次操作如下：找到权值最小边，如果权值为INFINFINF，结束循环；否则，将该边激活，查询当前图最大生成森林边权和的增量是否等于当前边权值，不等于说明遇到了环，不选该边，等于就更新lastlastlast为所查最大生成森林边权和，每次循环结束将所查边的权值设为INFINFINF。

最终图的最小生成森林边权和就是lastlastlast。

复杂度：O(m)O(m)O(m)

初始查询每条边的权值、检查加边后有无环，两部分代码都是O(m)O(m)O(m)

const int MAX_N = 510;

int n,m;
int w[MAX_N];

int query(string s){
	cout<<"? "<<s<<'\n';
	int x;
	cin>>x;
	return x;
}

int main(){
	freopen("1.in", "r", stdin);
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out", "w", stdout);
	cin>>n>>m;
	string s(m,'0');
	for(int i=0;i<m;i++){
		s[i]='1';
		w[i]=query(s);
		s[i]='0';
	}
	int last=0;
	while(1){
		int id=min_element(w,w+m)-w;
		if(w[id]==INF)
			break;
		s[id]='1';
		int now=query(s);
		if(now-last!=w[id])
			s[id]='0';
		else
			last=now;
		w[id]=INF;
	}
	cout<<"! "<<last<<'\n';
	return 0;
}
