D. Lost Arithmetic Progression

题意

给定两个等差数列B,CB,CB,C的首项、公差及项数，求等差数列AAA的种数，其中A∩B=CA\cap B=CA∩B=C

输入：t,b,q,y,c,r,z(1≤t≤100,−109≤b,c≤109,1≤q,r≤109,2≤y,z≤109)t,b,q,y,c,r,z(1\leq t\leq100,-10^9\leq b,c\leq10^9,1\leq q,r\leq 10^9,2\leq y,z\leq10^9)t,b,q,y,c,r,z(1≤t≤100,−109≤b,c≤109,1≤q,r≤109,2≤y,z≤109)

输出：种数无穷输出−1-1−1，否则输出种数(对109+710^9+7109+7取模)

思路

  1. 种数为000

    如果b1>c1 or bn<cnb_1>c_1\space or \space b_n<c_nb1​>c1​ or bn​<cn​，则CCC中有元素不在BBB中，也就不可能满足A∩B=CA\cap B=CA∩B=C

    如果dc%db≠0d_c\%d_b\neq0dc​%db​​=0，则CCC中有元素不在BBB中，也就不可能满足A∩B=CA\cap B=CA∩B=C

    如果(c1−b1)%db≠0(c_1-b_1)\%d_b\neq 0(c1​−b1​)%db​​=0，则B,CB,CB,C对不齐

  2. 种数无穷

    如果b1>c1−dc or bn<cn+dcb_1>c_1-d_c\space or\space b_n<c_n+d_cb1​>c1​−dc​ or bn​<cn​+dc​，则一定有一边AAA可以无限延展而不会和BBB再有交集，故种数无限

  3. 种数有限

    如果BBB包含了CCC的左右拓展项(即不满足2)，由于AAA是基于CCC拓展的，且BBB限制了AAA的拓展边界，则种数一定有限。

    计算种数：

    首先一定有dc=lcm(da,db)d_c=lcm(d_a,d_b)dc​=lcm(da​,db​)，才能保证A,BA,BA,B每隔dcd_cdc​会重合。

    因此我们只需要枚举dcd_cdc​的因数iii，判断其是否满足dc=lcm(i,db)d_c=lcm(i,d_b)dc​=lcm(i,db​)即可，若满足，则iii作为dad_ada​的贡献为(AAA基于CCC能往左拓展的个数+1+1+1)×\times×(AAA基于CCC能往右拓展的个数+1+1+1)，也即(dcda)2(\frac{d_c}{d_a})^2(da​dc​​)2。累计所有因数贡献就是最终答案

复杂度：O(rlog(r))O(\sqrt{r}log(\sqrt{r}))O(r​log(r​))

代码

#include <bits/stdc++.h>
#define int long long
#define INF INT_MAX
#define LINF LLONG_MAX 
#define endl '\n' 
using namespace std;
typedef long long ll;
typedef unsigned long long ull;
const int MAX_N = 1;
const int mod=1e9+7;

int T;
int b,q,y,c,r,z;
int ans;

void fn(int x){
	//如果da和db最小公倍数不是dc，则非法 
	if(x*q/__gcd(x,q)!=r)
		return ;
	//累加答案为(dc/da)^2 
	x=r/x;
	ans=(ans+x*x)%mod;
}

signed main(){
	freopen("1.in", "r", stdin);
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out", "w", stdout);
	cin>>T;
	while(T--){
		cin>>b>>q>>y>>c>>r>>z;
		ans=0;
		int bl=b,cl=c,br=b+(y-1)*q,cr=c+(z-1)*r;
		//答案不存在 
		if(bl>cl||br<cr||q>r||r%q>0||(cl-bl)%q>0){
			cout<<0<<'\n';
			continue;
		}
		//答案无穷多 
		if(bl>cl-r||br<cr+r){
			cout<<-1<<'\n';
			continue;
		}
		//统计答案 
		for(int i=1;i*i<=r;i++){
			//枚举A的公差da 
			if(r%i)
				continue;
			int j=r/i;
			//如果da合法，更新ans 
			fn(i);
			if(i!=j)
				fn(j);
		}
		cout<<ans<<'\n';
	}
	return 0;
}
