Codeforces Global Round 21 D. Permutation Graph 题解

D. Permutation Graph

题意

给定一个1…n1\dots n1…n的排列，构建一个有nnn个顶点的图，对于任意区间[l,r][l,r][l,r]，若ala_lal​是该区间的最小值/最大值，ara_rar​是该区间的最大值/最小值，则l,rl,rl,r连边，求从111到nnn的最短路径长度。

输入：t,n,a1…n(1≤t≤50000,1≤n≤250000,1≤ai≤n)t,n,a_{1\dots n}(1\leq t\leq50000,1\leq n\leq 250000,1\leq a_i\leq n)t,n,a1…n​(1≤t≤50000,1≤n≤250000,1≤ai​≤n)

输出：从111到nnn的最短路径长度

思路

结论：每次可以贪心的跳到最远可以到达的点，再继续往后跳，直到nnn结束，统计跳跃次数即可。

如何找最远可以到达的点？

**Case 1 : ai>ai+1a_i>a_{i+1}ai​>ai+1​ **

如果当前值不是最大，遍历i+1…ni+1\dots ni+1…n，找到比当前值小的最小值；如果当前值是最大，找i+1…ni+1\dots ni+1…n最小值。

**Case 2 : ai<ai+1a_i<a_{i+1}ai​<ai+1​ **

如果当前值不是最小，遍历i+1…ni+1\dots ni+1…n，找到比当前值大的最大值；如果当前值是最小，找i+1…ni+1\dots ni+1…n最大值。

其中，两种情况的第二种操作，都可以提前预处理两个数组ma,mima,mima,mi记录i…ni\dots ni…n的最大值和最小值。

代码

#include <bits/stdc++.h>
#define INF INT_MAX
#define LINF LLONG_MAX 
#define endl '\n' 
using namespace std;
typedef long long ll;
typedef unsigned long long ull;
const int MAX_N = 250010;

int T;
int n;
//ma[i]为ai~an最大值的编号 
//mi[i]为ai~an最小值的编号 
ll a[MAX_N],ma[MAX_N],mi[MAX_N];

int main(){
	freopen("1.in", "r", stdin);
	ios_base::sync_with_stdio(false);
	cout.tie(0);
	cin.tie(0);
	//freopen("1.out", "w", stdout);
	cin>>T;
	while(T--){
		cin>>n;
		for(int i=1;i<=n;i++)
			cin>>a[i];
		ma[n]=n,mi[n]=n;
		//更新ma和mi 
		for(int i=n-1;i>=1;i--){
			if(a[i]>a[ma[i+1]])
				ma[i]=i;
			else
				ma[i]=ma[i+1];
			if(a[i]<a[mi[i+1]])
				mi[i]=i;
			else
				mi[i]=mi[i+1];
		}
		//统计跳跃次数 
		int ans=0;
		//从1开始遍历 
		for(int i=1;i<n;){
			//如果下一个数是比当前大的 
			if(a[i+1]>a[i]){
				int j=i+1;
				//若当前值不是最小 
				if(mi[i]!=i){
					//遍历找到比ai大的最大值 
					for(int k=i+1;k<=n&&a[k]>a[i];k++)
						if(a[k]>a[j])
							j=k;
				}
				//若当前值是最小
				else
					//直接找i+1~n的最大值 
					j=ma[i+1];
				//更新i 
				i=j;
			}
			//如果下一个数是比当前小的 
			else{
				int j=i+1;
				//若当前值不是最大 
				if(ma[i]!=i){
					//遍历找到比ai小的最小值 
					for(int k=i+1;k<=n&&a[k]<a[i];k++)
						if(a[k]<a[j])
							j=k;
				}
				//若当前值是最大 
				else
					//直接找i+1~n的最小值
					j=mi[i+1];
				//更新i
				i=j;
			}
			//跳一次 
			ans++;
		}
		//输出跳跃次数 
		cout<<ans<<endl;
	} 
	return 0;
}
